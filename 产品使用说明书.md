
# 协同办公平台 - 产品使用说明书

## 一、产品概述

### 1.1 产品简介

协同办公平台是一款基于 Vue3 + Spring Boot 技术栈开发的企业级办公自动化系统，旨在提升组织内部的协作效率和信息流转速度。

### 1.2 主要功能模块

| 模块 | 功能描述 |
|------|----------|
| 用户认证 | 用户登录、权限管理 |
| 公文管理 | 公文的创建、编辑、发布、归档 |
| 流程审批 | 审批流程的发起、审核、驳回 |
| 会议管理 | 会议预约、会议室管理 |
| 知识文档 | 文档上传、分类、检索 |
| 督办监控 | 任务创建、跟踪、提醒 |
| 通知消息 | 系统通知、消息提醒 |

### 1.3 技术架构

- **前端**: Vue 3 + Vite + Vue Router + Axios
- **后端**: Spring Boot 2.7 + Spring Data JPA
- **数据库**: MySQL 8.0+
- **样式**: Font Awesome Icons

## 三、安装与部署

### 3.1 数据库配置

1. 创建数据库：
```sql
CREATE DATABASE ai CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 数据库连接信息：
   - 主机：localhost
   - 端口：3306
   - 数据库名：ai
   - 用户名：root
   - 密码：123456

### 3.2 后端部署

```bash
# 进入后端目录
cd backend

# 编译项目
mvn clean package

# 运行项目
mvn spring-boot:run
```

后端服务启动后访问地址：`http://localhost:8080`

### 3.3 前端部署

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 生产构建
npm run build
```

前端服务启动后访问地址：`http://localhost:5173`

### 4.2 主界面说明

登录成功后进入系统主界面，包含以下区域：

- **顶部导航栏**: 显示系统标题、通知消息、用户信息
- **左侧导航菜单**: 各功能模块入口
- **主内容区**: 当前模块的业务操作区域

### 4.3 功能模块使用说明

#### 4.3.1 公文管理

**功能说明：**
- 创建新公文：点击"新建公文"按钮，填写公文标题、内容、附件等信息
- 公文列表：查看所有公文，支持按状态筛选
- 公文详情：点击公文标题查看详细内容
- 公文操作：编辑、删除、发布、归档

#### 4.3.2 流程审批

**功能说明：**
- 发起审批：选择审批类型，填写审批内容
- 审批列表：查看待审批、已审批记录
- 审批操作：同意/驳回审批申请
- 审批历史：查看审批流程记录

#### 4.3.3 会议管理

**功能说明：**
- 预约会议：填写会议主题、时间、地点、参会人员
- 会议列表：查看所有会议安排
- 会议详情：查看会议具体信息
- 会议状态：取消会议、修改会议

#### 4.3.4 知识文档

**功能说明：**
- 上传文档：支持多种格式文件上传
- 文档分类：按类别管理文档
- 文档检索：关键词搜索文档
- 文档预览：在线查看文档内容

#### 4.3.5 督办监控

**功能说明：**
- 创建任务：设置任务标题、描述、负责人、截止日期
- 任务列表：查看所有督办任务
- 任务状态：进行中、已完成、逾期
- 任务跟踪：更新任务进度

#### 4.3.6 通知消息

**功能说明：**
- 消息提醒：未读消息数量提示
- 消息列表：查看所有未读通知
- 标记已读：单条或全部标记为已读

### 4.4 系统设置

**功能说明：**
- 用户管理：查看、添加、编辑、删除用户
- 角色管理：分配用户角色权限
- 部门管理：管理组织架构

## 六、常见问题

### 6.1 登录失败

**问题现象：** 输入用户名密码后无法登录

**可能原因：**
- 用户名或密码错误
- 数据库连接失败
- 后端服务未启动

**解决方案：**
- 检查用户名密码是否正确（测试账号：admin/123456）
- 确认MySQL服务已启动
- 确认后端服务已启动

### 6.2 页面显示空白

**问题现象：** 前端页面加载后显示空白

**可能原因：**
- 前端依赖未安装
- 端口被占用
- 网络请求失败

**解决方案：**
- 执行 `npm install` 安装依赖
- 检查端口5173是否被占用
- 打开浏览器开发者工具查看控制台错误

### 6.3 数据无法保存

**问题现象：** 操作后数据未保存到数据库

**可能原因：**
- 数据库连接配置错误
- 权限不足
- 数据格式错误

**解决方案：**
- 检查 `application.yml` 中的数据库配置
- 确认数据库用户权限
- 查看后端日志确认错误信息

### 6.4 通知消息不显示

**问题现象：** 点击铃铛图标后通知列表为空

**可能原因：**
- 后端服务未启动
- 网络请求失败
- 无未读消息

**解决方案：**
- 确认后端服务已启动
- 检查浏览器网络请求状态
- 使用数据库初始化数据
